Memory card

Se il prodotto viene fornito con una memory card rimovibile, la memory card è in genere compatibile con l'apparecchio acquistato ma potrebbe non esserlo con altri dispositivi o con le funzioni delle relative memory card. Prima dell'acquisto o dell'uso, verificare la compatibilità degli altri dispositivi. Se il prodotto è dotato di un lettore di memory card, prima dell'acquisto o dell'uso assicurarsi che la memory card sia compatibile.

La memory card viene formattata prima della consegna. Per riformattarla, occorre utilizzare un dispositivo compatibile. Non eseguire la procedura standard di formattazione del sistema operativo in uso per formattare la memory card su un PC. Per informazioni dettagliate, consultare le istruzioni del dispositivo o contattare Sony Call Center.
